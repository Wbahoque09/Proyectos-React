API= APPLICATION PROGRAMMING INTERFACE (INTERFAZ DE PROGRAMACIÓN DE APLICACIONES)
FUNCIONES, METODOS QUE OFRECE UNA LIBRERIA PARA SER UTILIZADA POR OTRO SOFTWARE COMO UNA CAPA DE ABSTRACCION
UNA API PONE A DISPOSICION RECURSOS QUE ESTAN ALOJADOS EN OTRO SERVIDOR O BASE DE DATOS
USUALMENTE HAY QUE ENVIAR UNA PETICION ESTRUCTURADA

¿QUE ES UNA API?
AL SER REACT UNA LIBRERIA QUE CORRE EN EL CLIENTE, NO PUEDE CONSULTAR UNA BASE DE DATOS, POR LO TANTO UNA API ES LA FORMA DE OBTENER DATOS DE UN SERVIDOR.
LA API PUEDE SER CREADA EN CUALQUIER LENGUAJE O FRAMEWORK: PYTHON, JAVA, NET CORE, EXPRESS, NODE.JS, LARAVEL, PHP
PARA ELLO DEBERA ENTREGAR UNA RESPUESTA TIPO JSON, USUALMENTE LAS API SON OBJETO JSON

¿COMO CONSULTAR UNA API?
AL SER JAVASCRIPT PUEDES UTILIZAR FETCH API Y OBTENER LOS DATOS PARA MOSTRARLOS EN PANTALLA
ALGUNAS API'S REQUIEREN UN KEY, Y OTRAS ESTAN PROTEGIDAS POR CORS
TAMBIEN ES POSIBLE INTEGRAR UNA LIBRERIA EXTERNA COMO AXIOS
