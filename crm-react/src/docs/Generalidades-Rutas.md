Outlet: El outlet en React Router es el espacio reservado en un componente donde se muestra el contenido de las rutas secundarias. Facilita el enrutamiento anidado y permite que el contenido de las rutas secundarias se renderice en el componente padre en el lugar designado por el outlet.  

Link: Se utiliza para crear enlaces (links) navegables dentro de una aplicación. Proporciona una forma de navegar entre las diferentes rutas definidas en tu aplicación sin tener que recargar la página completa.

useNavigate: es un hook de React Router que proporciona una forma programática de navegar entre las diferentes rutas de una aplicación web. Permite redirigir a una ruta específica mediante su uso en componentes funcionales de React, sin necesidad de utilizar la navegación tradicional basada en enlaces o botones.
